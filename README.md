# User Management System with SQLite

Система управління користувачами з використанням SQLite бази даних та Python. Проект включає повний функціонал для роботи з обліковими записами: додавання нових користувачів, оновлення паролів, автентифікацію з консольним введенням та безпечне зберігання паролів у вигляді SHA-256 хешів. Реалізовано валідацію вхідних даних, обробку помилок та захист від дублювання логінів.

## Структура та використання

Проект складається з трьох основних файлів: `main.py` (інтерактивне меню програми), `user_manager.py` (клас для роботи з базою даних) та `test_database.py` (автоматичне тестування всього функціоналу). Для запуску основної програми використовуйте `python main.py`, для тестування - `python test_database.py`. База даних `users_database.db` створюється автоматично при першому запуску та містить таблицю з полями: id, login, password (хеш), full_name та created_at.
